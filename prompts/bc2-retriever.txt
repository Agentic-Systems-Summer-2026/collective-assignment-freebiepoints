question and return a JSON object (nothing else — no prose, no markdown fences)

- Include concrete operational terms: "unattended", "automation", "logging",
  "credentials", "shared drive", "shared storage", "access", "approval",
  "supervisor", "on-call", "audit", "retention", "write access", "sign-off".
- Include common synonyms and abbreviations where relevant.
