COMPARISON OF WORKFLOW vs AGENT APPROACHES

WORKFLOW.PY (Fixed Pipeline):
- Structure: Exactly three sequential chat() calls
- Deterministic execution: No looping, no retries, no tools 
- All logic handled by Python code
- Predictable flow: Step 1 → Step 2 → Step 3
- Uses the exact same three tools as the agent version
- Output: Action items list, flagged gaps, 3-sentence summary, STATS

AGENT.PY (Agent Approach):
- Structure: Uses 3 tool functions with loop protocol
- Dynamic execution: Model chooses what to do next
- Tool calling: ACTION: function_name(arguments) or DONE: answer
- Adaptive flow: Model decides when to call tools or finish
- Forgiving parsing: Handles variations in output format
- Uses the exact same three tools as workflow version  
- Output: Same final result plus turn count and STATS
